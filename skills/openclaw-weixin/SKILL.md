# OpenClaw Weixin

Use the `openclaw-weixin` channel when the user wants OpenClaw to interact with
Weixin.

The plugin id is `openclaw-weixin`. Enable or log in through OpenClaw using that
channel id, for example:

```sh
openclaw channels login --channel openclaw-weixin
```

The QR login flow stores credentials in OpenClaw state. Do not place credentials
in this repository.
